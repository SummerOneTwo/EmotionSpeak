# EmotionSpeak 用户指南

## 1. 安装与环境配置
1. 克隆仓库并进入目录：
   ```bash
   git clone <repo-url>
   cd EmotionSpeak
   ```
2. 创建虚拟环境并激活：
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```
3. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

## 2. 启动项目
- 命令行运行：
  ```bash
  python main.py
  ```
- 或使用 Makefile：
  ```bash
  make run
  ```

## 3. 主要功能说明
- **文本情感分析**：输入文本，系统自动分析情感极性。
- **语音合成**：将文本转为带有情感色彩的语音。
- **音频播放**：播放合成的语音文件。
- **可视化**：生成词云和情感趋势图。

## 4. 测试与开发
- 运行全部测试：
  ```bash
  make test
  ```
- 代码格式化：
  ```bash
  make format
  ```
- 代码检查：
  ```bash
  make lint
  ```

## 5. 目录说明
- `src/`：核心代码
- `tests/`：单元测试
- `data/`：样例与输出
- `docs/`：文档
- `scripts/`：脚本

## 6. 常见问题
- 依赖安装失败：请检查 Python 版本与网络。
- TTS 无声音：请确认音频设备正常。
